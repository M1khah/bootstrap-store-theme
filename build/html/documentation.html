Documentation

Find out what's included in your Bootstrap theme and how to compile source files. Includes live examples and code snippets for every new component.

Gulpfile.js

You’ll need to install both Node and Gulp before using our included gulpfile.js

Once node is installed, run the following npm command to install Gulp.

$ npm install gulp -g

When you’re done, make sure you’ve installed the rest of the theme’s dependencies:

$ npm install

Theme source code

The scss/, js/,and fonts/directories contain the source code for our CSS, JS, and icon fonts. Within the scss/and js/directories you’ll find two subdirectories:

  • bootstrap/, which contains 3 version of Bootstrap.
  • custom/, which contains all of the custom components and overrides authored specifically for this theme.
The build/folder includes everything above, built into single CSS and JS files that can easily be integrated into your project.

Custom builds

scss/variables.scssis home to your theme’s variables. Note that your theme’s variablesfile depends on and overrides an existing Bootstrap variable file (found in scss/bootstrap/_variables.scss) and custom theme variables ( sass.custom/variables.scss).

Basic template

The basic template is a guideline for how to structure your pages when building with a Bootstrap Theme. Included below are all the necessary bits for using the theme’s CSS and JS, as well as some friendly reminders.

  
    <!DOCTYPE html>
    <html>
      <head>
        <meta charset="UTF-8"/>
        <meta name="viewport" content="width=device-width"/>
        <title>Bootstrap Theme Example</title>
        <link href="https://fonts.googleapis.com/css?family=Open+Sans:300,400,600" rel="stylesheet"/>
        <link rel="stylesheet" href="http://cdnjs.cloudflare.com/ajax/libs/highlight.js/9.9.0/styles/default.min.css"/>
        <link href="../css/style.css" rel="stylesheet"/></head>
      <body>
        <main class="container">
          <h1>Hello, world!</h1></main>
        <script src="https://use.fontawesome.com/67602d7308.js"></script>
        <script src="../../javascripts/jquery.js"></script>
        <script src="../../javascripts/bootstrap.js"></script>
        <script src="https://cdnjs.cloudflare.com/ajax/libs/highlight.js/9.9.0/highlight.min.js"></script>
        <script src="../../javascripts/doc.js"></script></body></html>
